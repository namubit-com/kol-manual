Login & Authentication

Security Notice: You will be authenticated with email address. Please do not disclose the site address and authentication number to anyone.

Login Process

Access the Login Page
Navigate to the NAMUBIT KOL login page. You'll see the NAMUBIT logo and language settings at the top.

Input your registered email address in the email field. This must be the email address associated with your KOL account.

Click the "NEXT" button to complete the login process and access the main dashboard.

KOL Management

Rate Configuration

Open Rate Setting Modal
Click the "Rate Setting" button on the dashboard to open the commission rate configuration modal.
Enter NAMUBIT ID
Input the 6-digit numeric NAMUBIT ID of the KOL you want to configure rates for.
Restriction: You can only enter 6 digits.
Configure Rates
Set commission rates for each exchange. The ME(%) is automatically calculated as Total(%) - KOL(%).
Exchange Total (%) ME (%) KOL (%)
Binance 25% Auto-calculated User Input
OKX 35% Auto-calculated User Input
Rate Error: The value of Total (%) cannot be exceeded.
Verification Code
Click "Send" to receive a verification code via email, then enter it to apply the rate changes.
Code Error: Invalid verification code. Please re-enter.

Troubleshooting

Common Issues

Login Problems
If you cannot receive verification codes, check your email spam folder and ensure your email address is correctly registered.
Rate Configuration Issues
Ensure KOL percentages don't exceed the Total percentage limits for each exchange.
Withdrawal Issues
Verify you meet the minimum withdrawal amount (10 USDT) and have sufficient balance including network fees.
Data Display Issues
If settlement or transaction data appears empty, try refreshing the page or selecting a different time period.

Error Messages

"You can only enter 6 digits."
NAMUBIT ID must be exactly 6 numeric digits. Remove any letters or extra numbers.
"The value of Total (%) cannot be exceeded."
KOL commission percentage cannot be higher than the Total percentage set for each exchange.
"Invalid verification code. Please re-enter."
The verification code may have expired or been entered incorrectly. Request a new code.

FAQ

General Questions

What is a KOL in NAMUBIT system?
KOL (Key Opinion Leader) is a user who can invite and manage other users, earning commissions from their trading activities across supported exchanges.
Which exchanges are supported?
Currently, NAMUBIT KOL system supports Binance and OKX exchanges, with more exchanges planned for future integration.
How are commissions calculated?
Commissions are calculated as: Total (%) - KOL (%) = ME (%). You set the KOL percentage, and your earnings (ME) are automatically calculated.
What is the minimum withdrawal amount?
The minimum withdrawal amount is 10 USDT, plus applicable network fees depending on the selected blockchain network.
How do I reset my verification code?
Click the "Send" button again to receive a new verification code. Each code expires after a certain time period for security purposes.
Can I change my KOL rates after setting them?
Yes, you can modify KOL commission rates at any time through the rate setting modal, but changes require email verification for security.
